# 四信号筛查规则

## 1. 具体收获 `concrete_takeaway`

检查读者能否复述至少一个具体概念、事实、数据、案例、方法或可检验观点。

- `strong`：能定位多个具体认知增量。
- `mixed`：有少量具体内容，但存在明显套话或重复。
- `weak`：主要留下情绪和模糊印象，难以复述具体内容。
- `unknown`：正文不足，或内容类型不适合当前判断。

不要因内容简单、面向初学者或篇幅短就直接判弱。

## 2. 来源具体性 `source_specificity`

检查是否出现可定位的人、研究、数据、原始链接、案例、时间或其他可复核材料。

- `strong`：来源可定位，且与重要论断对应。
- `mixed`：部分来源具体，关键结论仍有模糊跳跃。
- `weak`：大量使用“专家认为”“研究表明”但不给定位。
- `unknown`：内容类型本身不要求外部来源，或样本不足。

个人经历、艺术表达和纯观点不必按学术论文标准扣分。

## 3. 作者痕迹 `author_footprint`

检查是否有具体经历、独特判断、责任承诺、过程细节或与作者身份绑定的内容。

- `strong`：换账号或作者后，内容明显不再原样成立。
- `mixed`：一部分有作者痕迹，一部分是通用模板。
- `weak`：换号后可以原样发布。
- `unknown`：匿名、转述、转载或样本不足，无法判断。

匿名不是低质量证据；未知项必须保持未知。

## 4. 认知推进 `cognitive_progress`

检查内容是否减少不确定性、提供新角度、解释因果或提出可检验判断，而不是只刺激感动、愤怒、惊奇或焦虑。

- `strong`：产生可复述的新理解或下一步问题。
- `mixed`：情绪表达和认知推进并存。
- `weak`：主要刺激情绪，未看到认知增量。
- `unknown`：纯娱乐、文学、艺术或样本不足。

情绪内容可以有文化、社交或审美价值；不要把情绪本身判为无价值。

## 5. 风险聚合

v0.1 不使用百分比或加权分数。

只有同时满足以下条件，才允许 `slop_risk: high`：

1. 输入覆盖足够；
2. 至少三个信号为 `weak`；
3. 没有明显的内容类型例外；
4. 每个弱信号都有内容证据；
5. 输出明确声明这是疑似风险，不是真伪或作者动机结论。

否则使用 `medium` 或 `indeterminate`。

## 6. 内容类型例外

以下情况应优先保守：

- 诗歌、小说、讽刺、艺术表达：来源和作者痕迹可能不是核心质量标准。
- 个人经历：外部来源少不必然低质。
- 广告与营销：情绪诉求可能是任务本身，但仍可检查是否存在具体承诺和证据。
- 新闻、医学、法律、学术内容：本 Skill 不做事实或专业质量认证，应推荐来源核验和专家审查。
- 仅有标题、封面或短摘录：通常返回 `indeterminate`。
